# Logos de Clubes

Esta carpeta es para futuros logos que se usen en la aplicación de configuración.

## Naming Convention

**Los archivos deben nombrase igual que el nombre del club, pero:**
- Convertir a minúsculas
- Reemplazar espacios con guiones (`-`)

### Ejemplos:

- Club: `La Esquina` → Archivo: `la-esquina.png`
- Club: `Club de Pádel` → Archivo: `club-de-padel.png`
- Club: `Mi Club` → Archivo: `mi-club.jpg`

## Formatos soportados

- PNG
- JPG / JPEG
- SVG
- GIF
- WebP
